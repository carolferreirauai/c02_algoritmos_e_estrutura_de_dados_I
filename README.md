## 💻 Algoritmos I (C02) - Inatel

### 📌 Sobre o Repositório
Este repositório contém todos os códigos, listas de exercícios e trabalhos práticos desenvolvidos durante a disciplina **C02 - Algoritmos I**, cursada no **Instituto Nacional de Telecomunicações (Inatel)**.

O objetivo da disciplina é a introdução à lógica de programação e o desenvolvimento de algoritmos fundamentais.

* **Semestre:** 2024/2
* **Professor:** Daniel Mosca
* **Aluna:** Caroline Ferreira de Souza

### 🚀 Tecnologias Utilizadas
* **Linguagem:** C++
* **Ambiente de Desenvolvimento:** VS Code

### 💡 Como Executar
Para compilar e rodar os códigos deste repositório:
1. Clone o repositório: `git clone https://github.com/carolferreirauai/c02_algoritmosI.git`
2. Navegue até a pasta do arquivo desejado via terminal.
3. Compile o código: `g++ nome-do-arquivo.cpp -o executavel`
4. Execute o programa: `./executavel` (Linux/Mac) ou `executavel.exe` (Windows).
